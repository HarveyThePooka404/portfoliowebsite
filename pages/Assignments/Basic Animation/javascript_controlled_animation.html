04.02.01 Javascript controlled animation

Goal

Make a webpage where the user can turn ten different css animations for your character on and off.


This is the home page

Process

I had to find 10 visible CSS qualities that would impact the SVG of the character. The biggest challenge was to find a way to turn off one animation when a new one was started.

What worked well
